# 🌙 Midnight Aurora Proposal Card

A breathtakingly aesthetic, fully interactive, and hyper-optimized proposal/invitation card. Crafted with precision for the perfect moment.

## ✨ Features
- **GPU-Accelerated 3D Effects:** Hardware-accelerated CSS starry skies and twilight gradients that run flawlessly at 60FPS on any device.
- **Glassmorphism UI:** Stunning frosted glass aesthetics that dynamically blur the background.
- **Dynamic Poetic Responses:** The user interface reacts to interactions with gorgeous Bangla poetry and aesthetic GIFs.
- **Zero-Lag Celebration:** A heavily optimized canvas confetti engine for a spectacular, non-freezing victory screen.
- **Auto-Play Music:** Global interaction listeners to reliably stream Lofi/Aesthetic music exactly when the card is touched.

## 🚀 Built With
- **React 18**
- **Vite**
- **Tailwind CSS**
- **SweetAlert2**
- **Canvas Confetti**

## 🛠️ Usage
1. `npm install`
2. Drop your favorite `bangla-song.mp3` in the `/public/Music` directory.
3. `npm run dev` to start locally.
4. `npm run build` to compile the lightning-fast production bundle for Vercel/Netlify.
